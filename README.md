# dotfiles

macOSの設定ファイルや便利なスクリプトを管理するリポジトリです。

## 📁 構成

- **.claude/**: Claude AIの設定ファイルおよびプロンプトエンジニアリング用のルールファイル。
- **.raycast-scripts/**: [Raycast](https://www.raycast.com/) 用のスクリプトコマンド集。
  - 音声出力切り替えスクリプトなどが含まれています。
- **link.sh**: 設定ファイルのシンボリックリンクを自動作成するためのスクリプト。

## 🚀 セットアップ

このリポジトリをクローンした後、`link.sh` を実行してシンボリックリンクを作成します。

```bash
# リポジトリをクローン
git clone https://github.com/yuta/dotfiles.git ~/dotfiles

# ディレクトリへ移動
cd ~/dotfiles

# 実行権限を付与してセットアップスクリプトを実行
chmod +x link.sh
./link.sh
```

### link.sh の挙動について
`link.sh` は以下の動作を行います：
1.  ファイルの場合は、ホームディレクトリなどの所定の場所にシンボリックリンクを作成します。
2.  ディレクトリの場合は、ディレクトリ構造を維持したまま、配下のファイルごとに個別にシンボリックリンクを作成します。
3.  リンク先のディレクトリが存在しない場合は自動的に作成します。
