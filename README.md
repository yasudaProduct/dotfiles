# dotfiles

個人用のdotfiles管理リポジトリです。

## 概要

このリポジトリは、開発環境の設定ファイル（dotfiles）を管理し、複数のマシン間で一貫した環境を維持するためのものです。

## 構成

- `.claude/` - Claude AI関連の設定ファイル
  - `CLAUDE.md` - Claudeとの会話ガイドライン
- `link.sh` - dotfilesをホームディレクトリにシンボリックリンクするスクリプト

## セットアップ方法

### 1. リポジトリのクローン

```bash
git clone https://github.com/yasudaProduct/dotfiles.git
cd dotfiles
```

### 2. シンボリックリンクの作成

```bash
./link.sh
```

このスクリプトは、設定ファイルを適切な場所にシンボリックリンクとして配置します。既存のファイルがある場合は、`.bak`の拡張子を付けてバックアップされます。

## 使用方法

`link.sh`スクリプトは、`files_and_paths`配列で定義されたファイルとパスのペアに基づいて、自動的にシンボリックリンクを作成します。

新しいdotfileを追加する場合は、以下の手順を実行してください：

1. このリポジトリに設定ファイルを追加
2. `link.sh`の`files_and_paths`配列に、以下の形式でエントリを追加：
   ```bash
   "ソースファイルのパス:リンク先のパス"
   ```
3. `./link.sh`を実行してシンボリックリンクを作成
